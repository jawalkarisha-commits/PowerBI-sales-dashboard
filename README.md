# Sales Performance Dashboard | Power BI Project

## Overview
An interactive Power BI dashboard created to analyze sales performance, revenue trends, customer insights, and key business KPIs.

## Key Features
- KPI Tracking (Total Sales, Revenue, Profit)
- Monthly & Yearly Sales Trends
- Region-wise Performance Analysis
- Product Category Insights
- Customer Segmentation
- Dynamic Filters & Slicers
- Data Visualization for Business Decisions

## Tools Used
- Power BI
- Microsoft Excel
- Data Cleaning
- Data Visualization

## Business Insights
- Identified top-performing products
- Tracked monthly growth trends
- Compared region-wise sales contribution
- Monitored revenue and profit margins

## Files Included
- Dashboard Screenshots
- Power BI Report File (.pbix)
- Dataset (.xlsx / .csv)
